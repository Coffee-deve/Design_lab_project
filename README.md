The project focuses on the creation of the multimedia server, and also on the creation of the snake game.

## Snake game part:
Details about the snake game part.  
#### Pinout for TFT display:  
VCC ---> 3.3 V        
GND ---> GND  
CS  ---> D8  
RST ---> D6  
A0  ---> D1  
SDA ---> D7  
SCK ---> D5  
LED ---> 3.3 V  

#### Pinout for buttons
Buttons on one side:
D0 - 10k resistor -  Down
D2 - Right
D3 - Left
D4 - Up
All buttons on one side:
GND ---> GND
